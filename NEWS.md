# breedverse NEWS

## breedverse 0.1.0

### New Features

* Setup Golem framework structure
* Breedverse home description and initial structure
* Added **Qploidy** — ploidy estimation and dosage calling for polyploid species.
* Added **BIGapp** — an integrated app for genomic data analysis and visualization.
* Added **familia** — a tool for pedigree-based relatedness and family structure analysis.
* Added **AlloMate** — an application for allopolyploid genotype analysis and mate selection.
